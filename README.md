# ScrapingBNA Diario
Herramienta para obtener la cotización de divisas y billetes en forma directa desde la página web del *Banco de la Nación Argentina*


### Requerimientos

- Python >= 3
- [Scrapy]
- requests

### Run the script!

Ejecutar `./main.py` (o `python main.py`)
